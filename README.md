# engineering-lab
Personal homelab to explore systems engineering, networking, cloud technologies, and cybersecurity.

Welcome to my Engineering Lab.

This repository documents my journey as I develop practical engineering skills through hands-on projects and real-world experiments. Rather than studying technologies in isolation, I build complete systems to understand how software, networking, infrastructure, and security work together.

The goal of this project is to learn by building, breaking, fixing, and documenting every step of the process while preparing for a career in Computer Engineering.
